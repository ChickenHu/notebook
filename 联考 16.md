# 宏观
## 苔藓

|     | 苔                          | 藓                              |
| --- | -------------------------- | ------------------------------ |
| 配子体 | 多为扁平叶状体，无维管组织、单细胞假根        | 茎叶分化，有**中轴**无维管，单列细胞分支假根       |
| 孢子体 | 基足、短缩蒴柄、孢蒴、**弹丝**，成熟时不规则裂开 | 基足、长蒴柄、孢蒴、**蒴帽**（撕裂颈卵器）、**蒴齿** |
| 原丝体 | 不发达，不产生芽体                  | 发达，产生多个芽体，发育成独立的植物             |
