# 微观
## 几丁质

|      | α                | β            |
| ---- | ---------------- | ------------ |
| 排列方式 | 反向平行             | 平行           |
| 强度   | 高                | 低            |
| 来源   | 甲壳类动物、昆虫表皮、真菌细胞壁 | 软体动物、环节动物、硅藻 |

## CTAB 法
CTAB 即十六烷基三甲基溴化钠，为**阳离子去污剂**，可以在低离子强度溶液中沉淀核酸与酸性多聚糖，高离子强度下与蛋白质和多聚糖形成复合物，不能沉淀核酸，专克含多糖样品
