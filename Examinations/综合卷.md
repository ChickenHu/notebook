> 如图示，该米氏酶的双倒数作图反映了哪一种抑制剂的作用？
>![[米氏酶.excalidraw]]
>- [ ] A. 竞争性抑制剂
>- [ ] B. 反竞争抑制剂
>- [x] C. 竞争性抑制剂 + 非竞争性抑制剂
>- [ ] D. 非竞争抑制剂 + 反竞争抑制剂
>
>>[!important] 解析
>> 米氏方程
>> $$v=\frac{V_{max}[S]}{K_{m}+[S]}$$
>> 双倒数处理后
>> $$\begin{align} \frac{1}{v}&=\frac{K_m+[S]}{V_{max}[S]}\\&=\frac{K_m}{V_{max}}\times \frac{1}{[S]}+ \frac{1}{V_{max}}\end{align}$$
>> 由图可知纵截距增大，横截距增大，所以**表观**$V_{max}$ 减小，**表观**$K_m$ 增大，所以选 C
>>
>>>[!note] 抑制剂
>>> 竞争性抑制剂：竞争结合酶，$V_{max}$ 不变，**表观**$K_m$ 增大（作用可被添加底物抵消）
>>> 非竞争性抑制剂：直接抑制酶，$K_m$ 不变，$V_{max}$ 减小（作用可被添加酶抵消）
>>> 反竞争抑制剂：结合酶–底物中间物，$K_m$ 下降，$V_{max}$ 减小

>[!faq] (Not) Fun Facts About Sugar
>## 旋光性
>`$smiles=O=C(CO)CO`
> 二羟丙酮微微一笑 :smirk:
>## 稳定性
> 由于 `-CH=O` 的存在，在==碱性环境==下会被去质子化而形成不稳定的烯醇形式 `=C=O` 从而诱发一系列反应，例如变旋现象，异构反应，生成羧酸裂解重排等
>==酸性条件下==，相对会稳定一些，但是在强酸时仍然会发生脱水，裂解等反应
>## 糖胺聚糖
> 己糖醛酸 + 己糖胺 + 硫酸根
> 特别的，除素（葡糖胺）外其它的糖胺聚糖均含 N- 乙酰己糖胺，除硫酸角质素（D- 半乳糖）外均含有己糖醛酸，除透明质酸外均含有硫酸根

>[!faq] 果胶甲酯化
> 根据现有资料看来，高甲酯化果胶会因 `-COOH` 的缺失而无法结合 $Ca^{2+}$，但是却能在高 $[H^{+}]$ 时形成凝胶

>[!faq] 糖原贮积症
> 多且杂，基本上糖原代谢中涉及的所有酶都会引起，甚至 EMP 途径的酶（磷酸果糖激酶 -->Ⅶ型）也会引起。详见 [参考资料](https://www.msdmanuals.cn/professional/pediatrics/inherited-disorders-of-metabolism/glycogen-storage-diseases)。
> 此题中缺乏己糖激酶确实不会引起，因为它负责肝细胞中 `Glc` 磷酸化，跟糖原代谢沾不上边

>[!faq] 胆红素
> `$smiles=CC1=C(NC(=C1CCC(=O)O)CC2=C(C(=C(N2)C=C3C(=C(C(=O)N3)C)C=C)C)CCC(=O)O)C=C4C(=C(C(=O)N4)C=C)C`
> 血红素降解产物，卟啉开环 --> 高疏水，因此与清蛋白（疏水口袋，Site Ⅰ）结合运输
> 其结合可被脂肪酸，磺胺类药物，水杨酸，青霉素，吲哚美辛（非甾体抗炎药，可结合清蛋白另 Site Ⅱ）等竞争抑制
