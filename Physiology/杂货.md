## GLUT

| 亚型    | 分布                     | 功能                       |
| ----- | ---------------------- | ------------------------ |
| GLUT1 | 几乎所有                   | 基本                       |
| GLUT2 | 肝细胞，胰岛β细胞              | 双向转运，维持内外相同水平 `Glc`-->感受 |
| GLUT3 | 神经元                    | 转运                       |
| GLUT4 | 骨骼肌、心肌、脂肪组织            | 单向转运                     |
| GLUT5 | 小肠（顶端膜），脂肪组织、脑、肌肉（低水平) | `Fru` 转运                  |

## Na+/K+-ATPase

2 K+ in, 3 Na+ out （你怎么还没记住）

- 磷酸化（E1 态）：结合 Na+
- 去磷酸化（E2 态）：结合 K+
### 哇巴因

与 E2 状态下钠泵（β亚基）细胞外部结构结合
