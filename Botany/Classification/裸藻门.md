# 形态与结构
- 除胶柄藻属外，无细胞壁
- 有蛋白质构成周质体
- 具 9+2 鞭毛
- 有眼点：20 至 50 个橙色油滴，含有β- 胡萝卜素及其衍生物
- 有核，但分裂时核膜不消失，为原始的有丝分裂，故称中核
- 载色体：叶绿素 a，叶绿素 b，β- 胡萝卜素和 3 种叶黄素，3 层膜（2 层载色体膜和 1 层内质网膜），内有 3 条 1 束类囊体
- 裸藻淀粉，只在细胞质中
# 繁殖
- 细胞纵裂
- 胶质状态时
	- 失去鞭毛，分泌厚胶被 -->胶群体
	- 环境适宜时每个细胞形成新个体
	- 胞囊：度过恶劣环境
# 代表
- 裸藻属
- 柄裸藻属
# 地位

藻类和其他植物的祖先，与绿藻关系模糊不清
