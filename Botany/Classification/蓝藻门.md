# 形态与构造
- 原核生物
- 分为周质和中心质
- 有光合片层
- 叶绿素 a，藻蓝蛋白，藻红蛋白
- 光合产物：蓝藻淀粉，藻青素颗粒体
- 细胞壁 4 层
- 异形胞：固氮
# 繁殖
- 营养繁殖
	- 单细胞：直接分裂
	- 群体：细胞反复分裂，大群体分裂成小群体
	- 丝状体：形成藻殖段
- 无性繁殖
	- 丝状：产生厚壁孢子
	- 管胞藻目：外生孢子（不断向外分裂产生）/内生孢子（分裂成多个孢子后一并释放）
# 代表
- 单细胞或群体
	- 色球藻属
	- 微囊藻属
	- 管胞藻属
- 丝状体
	- 颤藻属
	- 念珠藻属
	- 鱼腥藻属：可形成水华
	- 真枝藻属：有真分枝
# 地位

和细菌最接近，与其他植物相差甚远，是独立的类群
