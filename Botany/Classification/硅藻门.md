# 形态构造
- 细胞壁：果胶质 + 硅质，分上壳（大）和下壳（小）
- 载色体：四层膜，类似金藻，色素为叶绿素 a，叶绿素 c，β- 胡萝卜素，==α- 胡萝卜素==，墨角藻黄素，硅藻黄素，硅甲藻黄素
- 同化产物：金藻昆布多糖，油
- 鞭毛：9+==0==
# 繁殖
- 细胞分裂，以老的半片作上壳，新的为下壳 -->子细胞只有一个等大，其他变小
- 复大孢子：用于恢复大小
	- 披针桥弯藻：2 细胞结合 -->减数分裂 -->4 个子核 -->退化 2 个 -->==2 个大小不等的配子== -->结合 -->2 个复大孢子 -->2 个新个体
# 代表
- 小环藻属
- 羽纹硅藻属
# 地位
- 与金藻，褐藻关系密切
- 细胞壁又类似于黄藻
- 设想是从含叶绿素 a 和 c，且具鞭毛的藻发育
